安置帮教工作流程

安置帮教工作流程

安置帮教的概念

刑满释放人员安置帮教工作，简称安置帮教工作，是在各级政府领导下，依靠各有关部门和社会力量对刑满释放人员进行的一种非强制性的引导、扶助、教育和管理活动。

安置帮教对象的范围

安置帮教工作对象是指刑满释放5年以内和接触社区矫正3年以内的人员。

安置帮教工作制度

（1）人员接收制度

收到《刑满释放人员通知书》或《解除矫正人员通知书》后应及时登记，7天内通知家人及亲属到监狱或劳教所接回。刑满释放人员回归后，7天内上门家访，了解刑满释放人员基本情况，发现问题，及时帮助解决，减少托管、失控现象发生。

（2）请示报告制度

安置帮教组织要建立请示报告制度，加强组织观念，对于工作中出现的重大问题应及时逐级上报，不得隐瞒不报，紧急情况要边处置边报告。上级部门对请示的问题要及时研究尽快予以批复。

（3）建档统计制度

司法所对安置帮教对象要逐人入档。安置帮教对象的犯罪情况、改造表现、家庭成员、社会关系、接受教育、考核鉴定等情况要记入档案。安置帮教组织要建立起安置帮教工作情况统计报表制度。统计报表和统计数据分析，要保证真实、准确，不得拒报、错报、漏报、虚报和瞒报。

（4）培训工作制度

安置帮教组织要建立安置帮教工作者的学习培训制度，采取多种形式对专业安置帮教工作者和社会志愿者进行业务培训，确保安置帮教工作的质量。

（5）信息报送制度

安置帮教组织要加强信息与报送工作，坚持客观性和实效性的原则，收集、整理本地区安置帮教工作的经验、做法、重要活动及典型案例等各种安置帮教工作信息，及时逐级上报。

（6）监督检查制度

司法行政机关、公安机关的安置帮教执行工作受人民检察院的监督。司法行政机关、公安机关对人民检察院的监督工作应予以配合。各级安置帮教组织要进一步加强自身建设，建立和完善内部监督检查制度。要积极接受纪律检查监察部门的监督和上级安置帮教组织的监督检查。

（7）情况通报制度

安置帮教工作各成员单位之间要建立情况通报和信息交流制度，至少每半年召开一次情况通报会，交流工作情况和工作信息，遇有重大政策出台或突发事件发生要随时通报。

（8）例会制度

帮教安置组织应当建立定期例会制度，传达上级安置帮教工作的指示精神，研究制定本地区安置帮教工作的规划和实施方案，听取有关部门的工作汇报，协调相关部门开展工作，研究解决试点工作中遇到的重大问题，强化“三项管理”，杜绝刑满释放人员的再犯罪。

作为一个特殊的群体，归正人员的心态很不平稳，存在过一天算一天的得过且过思想，为真正杜绝他们思想行为上的重复犯罪现象，我们在加强思想教育、化解消极心态和安置落实工作的基础上，重视对这一部分人员的跟踪监督管理工作。

安置帮教工作主要任务

（1）按照有关法律、法规和规章，依法对帮教对象进行管理和监督。

（2）通过多种形式，对帮教对象进行思想教育、法制教育、社会公德教育，帮教其不良心理和行为。使他们悔过自新，弃恶从善，成为守法公民。

（3）依法组织安置帮教对象参加适合其年龄、身体条件、劳动技能的社会公益劳动

（4）帮助帮教对象解决在生活、法律、心理等方面遇到的困难和问题。

安置帮教工作纪律

（1）不滥用职权、弄虚作假。

（2）不隐瞒案情，包庇、纵容帮教对象违法犯罪。

（3）不刑讯逼供或者体罚、虐待帮教对象。

（4）不玩忽职守，履行法定职责。

（5）不泄露帮教安置工作秘密。

（6）不利用帮教对象从事营利性活动或牟取其他权力。

（7）不得吃请或收受好处。
